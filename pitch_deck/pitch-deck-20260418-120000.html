THE PROTOCOL

Read what you sign — on the device itself.

Lucid is a clear-signing protocol for Solana programs.

Intent Definitions

Anchor IDL → human-readable ruleset. Every action templated.

Plain-English Signing

Signers approve via signMessage. Wallet renders on-device, outside any compromised host.

On-Chain Verify

Program reconstructs the message and verifies the signature. Tamper → reject.

Aligned with sRFC 38: Offchain Message Specification Version 1 — Anza, December 2025.

Speaker Notes
Lucid is a clear-signing protocol for Solana. Programs ship intents — which are templates generated from their IDL, each with human-readable text. Signers approve via signMessage so the wallet renders the action on the device, outside any compromised host. The on-chain program reconstructs the exact message and verifies the signature. Any tampering results in transaction rejection.
