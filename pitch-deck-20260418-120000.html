Speaker Notes
"Lucid introduces intent definitions — structured templates auto-generated from program IDLs that describe every permitted governance action in plain English. When a signer approves, their Ledger shows the action in human-readable text. The on-chain program reconstructs the expected message from state and verifies the ed25519 signature. If the transaction was tampered, the signature won't match."

Speaker Notes
"The core innovation — verifying that signers saw and approved a specific human-readable action — requires trustless signature verification. Solana's native ed25519 precompile makes this possible at near-zero cost. The program reconstructs the expected message from on-chain state, not from user input. That's the tamper-proof guarantee. No off-chain server, no trusted party."
